# 目录导航 - specs-tree-directory-optimization

此目录包含SDD（Specifications-Driven Development）系统中关于目录结构优化的相关文档。

## 项目概述
specs-tree-directory-optimization 是 OpenCode SDD Plugin 的一个特性，旨在将 `.sdd/.specs/` 目录结构优化为 `.sdd/specs-tree-root/` 结构。

## 目录结构
- spec.md - 功能需求规格说明
- plan.md - 技术实施计划  
- tasks.md - 详细任务分解
- state.json - 工作流状态跟踪
- TASK-*.completed - 任务完成标记文件
- *.md - 验证和过程记录文件

## 依赖关系
此功能依赖其他specs-tree-*系列功能模块，共同组成OpenCode SDD Plugin的完整能力。
