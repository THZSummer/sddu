## 📖 概述

创建一个自动化系统，用于生成和管理 .specs/ 目录中各种 README 模板，帮助开发者快速创建规范文档。

### 功能需求
1. 生成 .specs/ 目录的主 README.md
2. 创建 Feature 级别的 README 模板
3. 创建 Sub-Feature 级别的 README 模板
4. 支持自动生成和更新功能列表
