# ADR - 架构决策记录

## 📂 目录简介

存储 SDD 插件的架构决策记录 (Architecture Decision Records)。每个 ADR 文档记录一个重要的技术决策。

## 📁 目录结构

```
adr/
└── README.md          # 本文件
```

## 📄 文件说明

| 文件/目录 | 类型 | 说明 |
|-----------|------|------|
| README.md | 文件 | 本导航文件 |

## 📝 ADR 模板格式

```markdown
# ADR-XXX: [决策标题]

## 状态
[proposed | accepted | deprecated | superseded]

## 背景
[描述问题背景和上下文]

## 决策
[描述的决策内容]

## 后果
[描述决策带来的正面和负面后果]
```
