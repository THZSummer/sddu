# Architecture

## 📂 目录简介

存储 SDD 插件的架构决策记录 (ADR) 和技术架构文档。

## 📁 目录结构

```
architecture/
├── README.md          # 本文件
├── anti-skip-fix.md   # SDD 流程防跳过修复文档
├── adr/               # 架构决策记录
│   └── [ADR-XXX].md   # 具体 ADR 文档
└── archive/           # 历史档案
    └── [version].md   # 版本文档
```

## 📄 文件说明

| 文件/目录 | 类型 | 说明 |
|-----------|------|------|
| README.md | 文件 | 本导航文件 |
| anti-skip-fix.md | 文件 | SDD 流程防跳过修复文档 |
| adr/ | 目录 | 架构决策记录 (Architecture Decision Records) |
| archive/ | 目录 | 历史版本文档档案 |

## 📖 什么是 ADR

ADR (Architecture Decision Record) 是记录重要架构决策的文档，包括：
- 决策背景和上下文
- 考虑的备选方案
- 最终决策及理由
- 决策后果和影响
